Cancel Manufacturing Order

This application helps to automate process of Cancel Manufacturing Order.

Basic Features
  • This application provides facility to cancel work orders and cancel inventory moves while cancel manufacturing order, even if work order was finished and inventory moves was done or transferred.
  • "Cancel Work Orders", "Cancel Inventory Moves"
  • Company wise configuration.
